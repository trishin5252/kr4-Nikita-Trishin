 Пользователь может добавлять события (например, «Встреча с преподавателем», «Сдача КР») с указанием даты и времени, а приложение будет отображать их в виде списка — новое событие появляется сверху. Разработано React-приложение «Лента событий» для добавления записей с датой и временем. Пользователь вводит название события и выбирает дату через форму, после чего событие отображается в списке в обратном хронологическом порядке.
 Работа выполнялась через онлайн компилятор React.js Online Editor
